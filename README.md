# e-commerce project

Project for the [Teste-Inflr](https://github.com/GuilhermeGPG/Teste-Inflr) challenge.

E-commerce where users can either buy or sell products. 

### Installation
 
`$ pip install -r requirements.txt`

### Start server

```    
$ python3.4 manage.py runserver 8000
```
